# Windows setup

This project requires Python 3.13.x.

## 1. Create the environment

```bat
.\run.bat setup
```

## 3. Run the project

```bat
.\run.bat run
```

To run the concurrent mode:

```bat
.\run.bat run-concurrent
```

## 4. Useful commands

```bat
.\run.bat setup
.\run.bat run
.\run.bat run-concurrent
```
